# Assessment-Two-

## Name
ETO5513 Assessment 2: Team project

## Indroduction
This project contains files to produce a report on how different demographic factors influence a country’s level of access to clean fuels for cooking.
The report answers the following research questions:
    - Question 1: What is the relationship between GDP per capita and access to clean fuels?
    - Question 2: What is the relationship between population size and access to clean fuels?
    - Question 3: How does each region compare in access to clean fuels for cooking?
    - Question 4: Are there countries that have high access to clean fuels relative to their GDP per capita, and do they have anything in common?

The report produced contains sections written by Olivia Veal (Questions 1 and 2) and Marlies Smit (Questions 3 and 4), and an introduciton and conclusion written collaboratively.

## Usage
To produce this report run the file ETO5513-A2.Rmd.
Ensure you have the following packages installed:
    - tidyr
    - dyplyr
    - scales
    - ggplot2
    - knitr
    - bookdown
    - plotly
    - scales
Ensure your project has the following files:
    - Data/access-to-clean-fuels-for-cooking-vs-gdp-per-capita.csv (contains dataset)
    - references.bib (contains bibliography)
